# ValidaJá 📱

MVP de um aplicativo de controle de validade de produtos para pequenos mercados,
desenvolvido para a Sprint de **Gestão Ágil de Projetos e Produtos**.

**Aluno:** Edison Huarancca Chunga

## 📌 O problema

Pequenos mercados perdem dinheiro todo mês com produtos vencidos na prateleira.
O controle costuma ser feito em cadernos e planilhas, de forma manual e falha.

## 💡 A solução

O ValidaJá permite que o repositor escaneie o código de barras do produto,
registre a validade e receba alertas automáticos antes do vencimento —
dando tempo de colocar o produto em promoção e evitar a perda.
